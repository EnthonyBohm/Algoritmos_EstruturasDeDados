# LeetCode: Merge Intervals
Given an array of intervals(arrays), return an array that contain only intervals that don't overlap others, making the merge of those who have overlap
## Cases Tested

### Case 1
**Input**

[[1,3],[2,6],[8,10],[15,18]]

**Output**  
 [[1,6],[8,10],[15,18]]
---
### Case 2
**Input:**
[[1,4],[4,5]]

**Output**  
[[1,5]]

---
### Case 3
**Input:**

[[1,5],[10,15],[20,50],[1,30]]

**Output**  
[[1,50]]
